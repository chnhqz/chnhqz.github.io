# 第8章：支撑与阻力

## 引言

在股票技术分析中，**支撑（Support）**与**阻力（Resistance）**是最实用也最重要的概念之一。夏巴克指出："支撑与阻力水平是本课程中最有趣，也最实际的研究主题之一。"

支撑与阻力不仅帮助我们预测价格目标，还能提供明确的交易策略。本章将深入讲解支撑与阻力位的形成原理、预测方法以及在实战中的应用。

## 一、支撑与阻力的基本概念

### 1.1 定义

**阻力价位（Resistance）**：
- 股价上涨运动中被阻滞的价格区间
- 在该位置上股票的供给增加
- 也可被称作"供给区"

**支撑价位（Support）**：
- 股价下跌过程中股价停留的区间
- 在该位置上股票的需求增加
- 也可被称作"需求区"

### 1.2 角色转换原理

**关键规律**：
- 当一个由阻力形成的区域形态被突破后，它就变成了支撑位
- 当支撑价位被突破后，它成为以后股价运动的阻力位
- 支撑与阻力会不断交换身份

**原理说明**：
- 过去的底部变为未来的顶部
- 过去的顶部变成新的底部

## 二、支撑与阻力的形成原因

### 2.1 记忆因素的影响

**自然解释**：
- 公众等交易者倾向于记住并看重股价之前的走势
- 当股价在一个较窄的区域内波动了一段时间
- 公众的买卖交易都聚集在该区间内
- 因此人们容易记住这个特定的价位

**重要规律**：
- 股价停留在该区间内时间越久
- 其间发生的交易量越大
- 这个区间对于未来技术分析就越重要

### 2.2 人为因素的影响

**内部人操作**：
- 职业操盘者计划好的行为
- 清楚自己想要在什么价位吸入股票
- 在什么价位抛出股票

**操作策略**：
- 在100左右买入所需的大部分股票
- 决定在125左右卖出
- 第一次达到目标时先抛出约1/3的股票
- 让股价自己逐渐下跌
- 如果股价即将跌穿100，买回一些股票
- 新的上涨趋势再次接近125时，卖出剩余股票

### 2.3 可靠程度评估

**总体评价**：
- 支撑和阻力价位非常可靠
- 但并非万无一失
- 可能比之前学习的形态和法则更重要、更可靠

**失效情况**：
- 历史走势中预测的阻力位和支撑位可能不会持续有效
- 一旦这些价位被打破，就在反面意义上起作用
- 证明当前股价运动的巨大力量
- 表明这次运动将一直持续到下一次阻力或支撑位出现

## 三、长期趋势中的支撑与阻力

### 3.1 道琼斯工业平均指数案例

**1924-1929年牛市中的阻力位**：
- 120-124：停留5个月
- 160-164：停留1年多
- 200：停留6个月
- 320-324：停留5个月

**1929-1932年熊市中的支撑与阻力转换**：
- 196：1929年第一次恐慌性下跌的支撑位 → 1930年4月的阻力位
- 216：1928年早期上涨的阻力位 → 1930年的支撑位
- 160：1925-1927年的阻力位 → 1930年的支撑位 → 1931年的阻力位
- 120：1924年末的阻力位 → 1931年的支撑位 → 1931年的阻力位

### 3.2 通用电气公司案例

**1929-1932年熊市**：

**重要支撑位**：
- 42：1929年恐慌性下跌的支撑位
- 64-65：1929年末的阻力区 → 1930年的支撑位
- 54-55：1929年12月的支撑位
- 24-25：1931年的重要支撑位

**重要阻力位**：
- 78-79：1930年阻力，持续一个半月
- 94-96：1930年阻力位

**1932-1936年牛市中的转换**：
- 18：1932年上涨刺穿 → 1933-1934年持续影响
- 24-25：1931年的支撑 → 1933-1935年的阻力
- 42：1929年的支撑 → 1936年的阻力

## 四、中期趋势中的支撑与阻力

### 4.1 微小顶部与底部的转换

**典型案例：斯奈德包装公司**

**上涨过程中的支撑现象**：
- 7-8月：19遇到阻力 → 8月29日突破
- 反弹在21形成微小顶部 → 回落至19得到支撑
- 涨至23½ → 跌至21得到支撑
- 每次小幅下跌都在前面小幅上涨的顶部得到支撑

**反转后的支撑与阻力**：
- 11月中期反转后
- 第一次剧烈下跌停滞在10月末微小顶部附近
- 第二次下跌在9月最后一周的顶部附近得到支撑

### 4.2 判断标准

**对于中期趋势交易者**：
- 上涨过程中的微小支撑位被突破 → 没有负面意义
- 如果突破比较重要的支撑位 → 技术形势将发生重要变化
- 中期趋势很可能被反转

## 五、形态中的支撑与阻力

### 5.1 头肩形态

**颈线的作用**：
- 突破点之后的颈线是走势图中非常关键的一条线
- 颈线第一次被突破表明趋势的反转
- 之后股价可能会回到颈线附近
- 但极少能显著地穿过

**倾斜颈线**：
- 如果颈线呈水平，支撑或阻力水平容易解释
- 如果颈线向上或向下倾斜，功能更类似于趋势线

### 5.2 矩形和直角三角形

**矩形形态**：
- 上下边界分别代表阻力和支撑水平
- 突破后角色立即交换
- 上涨突破后，上边界变为支撑水平
- 下跌突破后，下边界变为阻力水平

**直角三角形**：
- 上升三角形的上边线（水平边线）被突破后成为支撑线
- 下降三角形的下边线被突破后成为阻力线

### 5.3 对称三角形

**突破后的支撑与阻力**：
- 取决于突破发生时距离理论端点的距离
- 早期突破：形态边线的延长线成为支撑/阻力线
- 端点附近突破：端点水平成为支撑/阻力线

**三种情况**：

**情况一：早期突破后很快折返**
- 回撤将在形态边线的延长线附近停住
- 向上突破：上边线成为支撑线
- 向下突破：下边线成为阻力线

**情况二：突破后运动一段距离才折返**
- 回撤一般在对称三角形的端点水平附近遇到阻力

**情况三：运动相当长距离后才掉头**
- 折返会停在三角形刚形成时的边线顶点水平上
- 股价已经离开了对称三角形的"磁场"区域

### 5.4 支点（Cradle）

**定义**：
- 两条趋势线相交时形成的交点
- 根据形状特征被称作"支点"

**特点**：
- 形成更大的阻力
- 股价并不总是会折返到支点
- 但当这种情况真的出现时
- 股价运动总会在该点停止并掉头

## 六、缺口中的支撑与阻力

### 6.1 突破缺口

**特点**：
- 形成缺口的股价运动建立了一道支撑或阻力"防线"
- 阻挡之后的股价回到旧形态

**支撑/阻力水平**：
- 上升突破缺口：缺口顶部附近水平得到支撑
- 下跌缺口：形成阻力水平

### 6.2 调整缺口

**特点**：
- 有时也具有支撑或阻力作用
- 如果缺口比较宽，顶部水平成为支撑
- 如果顶部水平支撑失败，中间或底部可能形成支撑
- 如果这些都失败，趋势将持续到下一个支撑水平

## 七、特殊价位的意义

### 7.1 50、100等价位的特殊意义

**心理作用**：
- 整数关口往往具有心理支撑或阻力作用
- 50、100等整数价位尤其重要
- 交易者往往在这些价位设置买入或卖出指令

### 7.2 中期运动的半途理论

**原理**：
- 股价运动往往在半途位置遇到支撑或阻力
- 可以帮助预测价格目标

## 八、支撑与阻力的实战应用

### 8.1 买入或卖出的第二次机会

**应用场景**：
- 错过第一次操作的机会
- 在仍然有利可图的水平上进行补操作

**案例**：
- 股价上涨至突破形态时，由于某种原因没有买入
- 股价回跌至形态支撑线附近
- 获得第二次买入的良机

### 8.2 避免过分担心

**应用场景**：
- 买入股票后看到股价回落到形态边线附近
- 通过支撑与阻力分析知道这是正常现象
- 只要股价在边线处得到支撑，对之前的判断仍然可以保持信心

### 8.3 卖空操作中的应用

**策略**：
- 在阻力位附近卖空
- 在支撑位附近平仓
- 设置止损位于阻力位的另一侧

## 九、本章总结

本章我们深入学习了支撑与阻力：

1. **基本概念**：
   - 阻力：上涨中被阻滞的价格区间
   - 支撑：下跌中停留的价格区间
   - 角色转换：突破后支撑变阻力，阻力变支撑

2. **形成原因**：
   - 记忆因素：公众对历史价位的记忆
   - 人为因素：内部人有计划的操作

3. **长期趋势中的应用**：
   - 过去的底部变为未来的顶部
   - 过去的顶部变成新的底部
   - 周线图比月线图更准确

4. **形态中的应用**：
   - 头肩形态：颈线的作用
   - 矩形和直角三角形：边线的支撑阻力
   - 对称三角形：三种情况的分析
   - 支点：两条趋势线的交点

5. **实战应用**：
   - 提供第二次操作机会
   - 避免过分担心正常回撤
   - 卖空操作中的应用

**核心原则**：
- 支撑与阻力是非常可靠的技术工具
- 但并非万无一失
- 需要结合成交量和其他技术因素综合判断
- 一旦被突破，就在反面意义上起作用
