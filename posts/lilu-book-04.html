第4讲：价值投资在中国的展望 ​

投资的大道上却根本没人，交通一点都不堵塞，冷冷清清。人都去哪了呢？旁门左道上车水马龙！

引言：为什么开这门课 ​

2015年10月，李录在北京大学光华管理学院开设了国内第一门价值投资课程。

背景：

  • 刚经历股灾（2015年6-8月）
  • 市场从牛市到熊市只用了3个月
  • 无数投资者损失惨重
  • 价值投资理念在中国几乎空白

李录的四个核心问题：

  1. 资产管理行业的独特性及其道德底线要求
  2. 从长期看，哪些金融资产可以持续增长？
  3. 什么是投资的大道？如何成为优秀投资人？
  4. 价值投资在中国是否适用？

第一部分：资产管理行业的道德底线 ​

行业的两个独特性 ​

独特性一：用户无法判断产品好坏 ​

与其他行业的对比：

行业用户判断方式
汽车驾驶体验
餐饮用餐体验
酒店入住体验
服装穿着体验
资产管理几乎无法判断

为什么无法判断？

  1. 时间滞后：需要5-10年才能判断
  2. 结果混杂：运气与能力难以区分
  3. 信息不对称：用户不懂投资逻辑
  4. 业绩波动：短期业绩充满噪音

后果：

正是因为没有办法判断（产品和服务的优劣），所以绝大部分理论都和屁股决定脑袋有关。

独特性二：报酬与贡献脱节 ​

收费模式的问题：

收费方式问题
按比例收费不管盈亏都收费
2+20模式赚钱提成，亏钱不赔
指数基金对比业绩差也能收费

结果：

  • 行业报酬远高于其他行业
  • 实际贡献往往有限
  • 定价反映从业者利益，而非客户利益

两条道德底线 ​

底线一：追求真知是道德责任 ​

把对真知、智慧的追求当作是自己的道德责任，要有意识地杜绝一切屁股决定脑袋的理论。

具体要求：

  1. 持续学习，追求真理
  2. 不被利益左右判断
  3. 不散布对自己有利的谬论
  4. 不被似是而非的理论蛊惑

为什么重要？

投资行业充满：

  • 似是而非的观点
  • 屁股决定脑袋的理论
  • 复杂的判断和预测

如果没有追求真知的责任感，很容易自欺欺人，最终害人害己。

底线二：建立受托人责任 ​

客户给你的每一分钱，你都把它看作是自己的父母辛勤劳动、勤俭节省、积攒了一辈子、交到你手上去打理的钱。

受托人责任（Fiduciary Duty）的内涵：

  • 把客户利益置于自身利益之上
  • 像对待父母的钱一样对待客户的钱
  • 建立长期的信任关系
  • 为客户创造真正的价值

先天基因论：

受托人责任这个概念，我认为多多少少有些先天的基因在里面。我所了解的人里或者有这个基因，或者没有这个基因。

警示：

没有这种基因的人，以后无论用什么方式，基本上都没有办法让他有。如果你的钱交到这些人手里，那真是巨大的悲剧。

第二部分：长期财富增长的真相 ​

200年资产回报数据 ​

数据来源：宾州大学沃顿商学院西格尔教授（Jeremy Siegel）

美国1802-2022年大类资产表现：

资产类别200年后价值（扣除通胀）年化回报
现金0.05美元-1.4%
黄金3.12美元0.5%
短期债券275美元2.7%
长期债券1,600美元3.6%
股票1,030,000美元6.7%

图7：美国自1801年至今大类资产的回报表现（来源：Siegel, Jeremy, Future for Investors, 2005）

为什么差距如此巨大？ ​

原因一：通货膨胀 ​

现金的贬值：

  • 200年平均通胀率：1.4%/年
  • 1美元 → 0.05美元（购买力）
  • 丢失了95%的价值

数学解释：

(1 - 0.014)^200 ≈ 0.06

原因二：经济增长 ​

GDP增长：

  • 200年增长约33,000倍
  • 年化增长率约3%

股票与GDP的关系：

  1. GDP增长 → 公司销售额增长
  2. 销售额增长 → 利润增长（成本相对固定）
  3. 利润增长 → 股票价值增长
  4. 平均市盈率15倍 → 每股收益约6.7%

复利的力量：

爱因斯坦把复利称之为世界第八大奇迹是有道理的。

1.067^200 ≈ 1,030,000

第三部分：价值投资的四大理念 ​

理念一：股票是公司的部分所有权 ​

核心观点：

股票不仅仅是可以买卖的证券，实际上代表的是对公司所有权的证书，是对公司的部分所有权。

为什么重要？

视角理解
投机者股票是交易筹码
价值投资者股票是公司所有权

正道 vs 邪道：

  • 正道：投资帮助公司增长，分享应得的收益
  • 邪道：零和博弈，从他人损失中获利

可持续性的检验：

如果把赚钱的方法一点一滴毫无保留告诉所有的人，大家都觉得你这个赚钱的方法真对，真好，我佩服，这就是可持续的。这就叫大道，这就叫正道。

理念二：市场是为投资者服务的 ​

核心观点：

市场的存在只是为你而服务的。能够给你提供机会，让你去购买所有权，也会给你个机会，在你很多年之后需要钱的时候，能够把它出让，变成现金。

正确理解市场：

错误理解正确理解
市场是老师市场是工具
市场总是对的市场经常犯错
跟随市场利用市场
价格=价值价格≠价值

与95%市场参与者的区别：

这个市场从来都不能告诉你，真正的价值是什么。它只能告诉你的只是价格是什么，你不能把市场当作你的老师，你只能把它当作一个可以利用的工具。

理念三：安全边际 ​

核心观点：

投资的本质是对未来进行预测，而预测得到的结果不可能百分之百准确，只能是从零到接近一百。那么当我们做判断的时候，就必须要预留很大的空间，叫安全边际。

为什么需要安全边际？

  1. 预测的不确定性：没有人能100%预测未来
  2. 错误的缓冲：即使预测错误，也不会损失太多
  3. 正确时的超额回报：预测正确时，回报更高

实践要求：

你的买入价格一定要大大低于公司的内在价值。

安全边际的计算：

安全边际 = 内在价值 - 市场价格

理念四：能力圈 ​

巴菲特的独特贡献：

投资人可以通过长期不懈的努力，真正建立起自己的能力圈，能够对某些公司、某些行业获得比几乎所有人更深的理解，而且能够对公司未来长期的表现，做出比所有其他人更准确的判断。

能力圈的核心：边界

没有边界的能力就不是真的能力。如果你有一个观点，你必须要能够告诉我这个观点不成立的条件，这时它才是一个真正的观点。

市场先生的作用：

市场存在的目的是什么？对于市场参与者而言，市场存在的目的就是发现人性的弱点。

真正的风险：

只有在这个意义上投资才真正是有风险的，这个风险不是股票价格的上上下下，而是资本永久性地丢失，这才是真正的风险。

第四部分：为什么价值投资是大道却人迹罕至？ ​

大道的特征 ​

为什么价值投资是大道？

  1. 逻辑简单清晰：四个理念，易于理解
  2. 经过长期验证：巴菲特57年业绩
  3. 可持续可复制：方法可以传授学习
  4. 多方共赢：帮助公司、客户、自己

为什么走的人很少？ ​

原因一：太慢 ​

康庄大道非常慢。听起来能走到头，但实际上很慢。

时间成本：

  • 买入后可能长期不涨
  • 市场不理性的时间可能很长
  • 需要等待价值回归

原因二：太难 ​

预测未来的困难：

投资的本事是对未来进行预测，真正要理解一个公司、一个行业，要能够去判断它未来5年、10年的情况。

能力圈建立的难度：

  • 需要多年不懈努力
  • 需要刻苦学习
  • 圈开始时非常狭小

原因三：短期诱惑 ​

市场总是让你感觉到短期可以获利——你短期的资产确确实实可以有巨大的变化，所以这会给你幻觉，让你想象在短期里可以获得巨大利益。

旁门左道的吸引力：

  • 短期暴利的可能性
  • 成为"股神"的幻觉
  • 快速致富的梦想

原因四：需要诚实 ​

做这个行业最根本的要求，是一定要在知识上做彻彻底底、百分之百诚实的人。

自欺欺人的陷阱：

  • 屁股决定脑袋
  • 假话说多了自己都信
  • 难以承认错误

时间的要求 ​

在这个行业里，要在很长时间持续不断地艰苦地工作，才有可能真正地成才，这个时间恐怕要15年以上。

成才的标准：

  • 连续15年以上优异成绩
  • 在正确的道路上研究
  • 能力远远大于运气

第五部分：价值投资在中国是否适用？ ​

核心问题 ​

质疑的声音：

  • 中国是不是特殊？
  • 价值投资是否只适合美国？
  • 中国市场是否太投机？

李录的回答 ​

第一，3.0文明的普适性：

3.0文明的本质是什么呢？就是在这个时期，由于种种的原因，出现了现代科技和自由市场经济，这两者的结合形成了我们看到的3.0文明。

第二，中国已经在市场体系中：

中国在1990年代以后，就已经全面进入到全球大的自由市场体系里了。

第三，价值投资的基本逻辑不变：

  • 股票代表公司所有权 ✓
  • 市场可以为你所用 ✓
  • 需要安全边际 ✓
  • 需要能力圈 ✓

中国的特殊性 ​

确实存在差异：

  • 市场成熟度不同
  • 投资者结构不同
  • 监管环境不同
  • 文化背景不同

但核心机制相同：

  • 经济持续增长
  • 优秀公司创造价值
  • 价格围绕价值波动
  • 长期看价值回归

第六部分：实战应用与常见陷阱 ​

建立正确的投资流程 ​

第一步：道德自律

  • 追求真知
  • 建立受托人责任

第二步：资产配置

  • 长期看，股票是最佳资产
  • 避免现金的长期持有

第三步：选择投资方法

  • 走价值投资大道
  • 避开旁门左道

第四步：建立能力圈

  • 选择能理解的行业
  • 深入研究少数公司
  • 明确能力边界

第五步：严格执行

  • 等待安全边际
  • 长期持有
  • 不理会市场波动

常见陷阱 ​

陷阱一：短期业绩焦虑 ​

表现：

  • 频繁查看账户
  • 被短期波动影响
  • 追逐热点

解决：

  • 降低查看频率
  • 关注长期价值
  • 建立投资纪律

陷阱二：能力圈幻觉 ​

表现：

  • 认为自己什么都懂
  • 频繁更换投资领域
  • 追逐热门行业

解决：

  • 承认无知
  • 专注少数领域
  • 明确说"不"

陷阱三：安全边际不足 ​

表现：

  • 追高买入
  • 过度乐观
  • 忽视风险

解决：

  • 保守估值
  • 耐心等待
  • 分散投资

陷阱四：缺乏耐心 ​

表现：

  • 频繁交易
  • 过早卖出
  • 追逐短期机会

解决：

  • 降低交易频率
  • 延长持有期
  • 相信复利

本章总结 ​

核心要点回顾 ​

  1. 行业道德底线：追求真知、受托人责任
  2. 长期资产表现：股票200年增值100万倍
  3. 价值投资四理念：所有权、市场工具、安全边际、能力圈
  4. 大道人迹罕至：太慢、太难、短期诱惑、需要诚实
  5. 中国同样适用：3.0文明普适，中国已在市场体系中

关键公式 ​

  • 复利公式：时间 × 收益率 = 巨大差异
  • 安全边际：内在价值 - 市场价格
  • 能力圈边界：能说清楚什么情况下观点错误

关键术语 ​

  • 受托人责任：把客户利益置于自身之上的道德义务
  • 安全边际：预留的预测错误缓冲空间
  • 能力圈：通过长期努力建立的深度理解领域
  • 市场先生：市场的非理性特征比喻

延伸思考 ​

  1. 如何检验自己是否有受托人责任的基因？
  2. 你的能力圈边界在哪里？
  3. 为什么明知是大道，却难以坚持？
  4. 在中国市场实践价值投资的具体挑战？
